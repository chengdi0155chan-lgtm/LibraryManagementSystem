# 📚 图书馆管理系统

## 项目简介
基于 Spring Boot 的图书馆管理系统，实现图书的入库、借阅、归还、查询，以及用户管理、借阅统计等基础功能。

## 🎯 作业目标
1. 掌握软件工程的核心流程（需求分析、设计、编码、测试、部署）
2. 学会使用AI工具辅助代码编写、调试与优化
3. 掌握代码质量评价的核心维度与方法

## 🛠️ 技术栈
- **后端框架**: Spring Boot 3.2.5
- **数据库**: MySQL 8.0
- **数据持久层**: Spring Data JPA
- **API文档**: OpenAPI 3.0 (Swagger)
- **安全框架**: Spring Security
- **构建工具**: Maven
- **开发工具**: Eclipse 2025-12
- **Java版本**: JDK 17

## 📋 功能模块
1. **用户管理模块**
   - 用户注册、登录、权限管理
   - 角色分配（管理员、图书管理员、普通用户）
   - 用户状态管理

2. **图书管理模块**
   - 图书信息增删改查（CRUD）
   - 图书分类管理
   - 库存管理（总册数、可借册数）

3. **借阅管理模块**
   - 图书借阅、归还、续借
   - 借阅记录查询
   - 超期罚款计算

4. **统计报表模块**
   - 用户统计
   - 图书统计
   - 借阅统计
